# 移动会议单网页应用设计文档

## 1. 项目概述

### 1.1 项目名称
移动会议管理

### 1.2 项目目标
创建一个轻量级的单页面应用，用于管理移动会议信息，支持从剪贴板自动解析会议信息，一键快速入会。

### 1.3 技术约束
- 单 HTML 文件，本地直接运行
- 不依赖第三方库（纯 HTML/CSS/JavaScript）
- 现代浏览器兼容（Chrome, Safari, Edge, Firefox）
- 响应式设计，适配桌面和移动设备

#### 3.2.3 主题切换实现

```javascript
// 主题管理
const ThemeManager = {
  STORAGE_KEY: 'meeting-theme-preference',

  init() {
    const saved = localStorage.getItem(this.STORAGE_KEY);
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;

    if (saved === 'dark' || (saved === 'auto' && prefersDark)) {
      document.documentElement.setAttribute('data-theme', 'dark');
    }

    // 监听系统主题变化
    window.matchMedia('(prefers-color-scheme: dark)').addEventListener('change', (e) => {
      if (localStorage.getItem(this.STORAGE_KEY) === 'auto') {
        document.documentElement.setAttribute('data-theme', e.matches ? 'dark' : 'light');
      }
    });
  },

  toggle() {
    const current = localStorage.getItem(this.STORAGE_KEY) || 'auto';
    const modes = ['light', 'dark', 'auto'];
    const next = modes[(modes.indexOf(current) + 1) % modes.length];

    localStorage.setItem(this.STORAGE_KEY, next);

    if (next === 'auto') {
      const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
      document.documentElement.setAttribute('data-theme', prefersDark ? 'dark' : 'light');
    } else {
      document.documentElement.setAttribute('data-theme', next);
    }

    return next;
  }
};
```

### 6.2 正则解析函数

```javascript
function parseMeetingText(text) {
  const patterns = {
    title: /会议主题[:：]\s*(.+)/,
    // 支持标准格式和简写格式（如：3月13日9:00）
    time: /会议时间[:：]\s*(\d{4}-\d{2}-\d{2}\s+\d{2}:\d{2}|\d{1,2}月\d{1,2}日\s*\d{1,2}:\d{2})/,
    // 支持"移动会议"和"会议号"两种格式
    meetingNumber: /(?:移动会议|会议号)[:：]\s*(\d+)/,
    password: /密码[:：]\s*(\d+)/,
    link: /参会链接[:：]\s*(https?:\/\/\S+)/,
    organizer: /(.+?)邀请您参加/
  };

  const result = {};
  for (const [key, pattern] of Object.entries(patterns)) {
    const match = text.match(pattern);
    result[key] = match ? match[1].trim() : '';
  }

  // 处理简写时间格式（如：3月13日9:00 → 2026-03-13 09:00）
  if (result.time && !result.time.includes('-')) {
    result.time = normalizeTimeFormat(result.time);
  }

  return result;
}

// 将简写时间格式转换为标准格式
function normalizeTimeFormat(timeStr) {
  const now = new Date();
  const year = now.getFullYear();
  // 匹配 "3月13日9:00" 或 "03月13日 09:00" 等格式
  const match = timeStr.match(/(\d{1,2})月(\d{1,2})日\s*(\d{1,2}):(\d{2})/);
  if (match) {
    const [, month, day, hour, minute] = match;
    return `${year}-${month.padStart(2, '0')}-${day.padStart(2, '0')} ${hour.padStart(2, '0')}:${minute}`;
  }
  return timeStr;
}
```

### 6.3 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| 移动端 | < 640px | 单列卡片，全宽 |
| 平板 | 640px - 1024px | 双列网格 |
| 桌面 | > 1024px | 三列网格，最大宽度 1200px |

## 7. 错误处理

### 7.1 错误类型及处理

| 错误场景 | 处理方式 | 用户提示 |
|---------|---------|---------|
| 剪贴板权限拒绝 | 引导用户手动粘贴 | "请手动粘贴会议信息到添加表单" |
| 解析失败 | 显示原始文本，允许手动编辑 | "无法自动识别，请检查内容" |
| 存储已满 | 清理旧数据或导出备份 | "存储空间不足，请删除旧会议" |
| 链接无效 | 入会前验证 | "会议链接格式不正确" |
| 时间解析错误 | 使用当前时间作为默认值 | "时间格式无法识别，请手动修改" |

## 8. 性能考虑

### 8.1 优化策略
- **防抖处理**: 剪贴板检测使用防抖，避免频繁读取
- **虚拟滚动**: 会议数量超过50个时使用虚拟滚动
- **懒加载**: 会议详情按需展开
- **状态缓存**: 会议状态计算结果缓存1分钟

### 8.2 存储限制
- LocalStorage 限制约 5-10MB
- 单个会议数据约 500 字节
- 支持约 10000 个会议记录
